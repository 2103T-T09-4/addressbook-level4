1. Introduction

The Car Park Finder is an application to find HDB (Housing Development Board) car parks in Singapore. It allows you to work with a Command Line Interface (CLI) to display a list of car parks with the convenience of simply typing. If you use the computer frequently and commute by driving, you would find our application useful in helping you obtain various information about car parks.

2. Quick Start

  1. Ensure you have Java version 9 or later installed in your Computer.

  2. Download the latest carparkbook.jar here.

  3. Copy the file to the folder you want to use as the home folder for your Car park book.

  4. Double-click the file to start the app. The GUI should appear in a few seconds.

  5. Type the command in the command box and press Enter to execute it.

  6. Some example commands you can try:

    • list: list your favorites list, with all the car park details

    • help: shows the list of available commands and how to use them

    • exit: exit the program

  7. Refer to Section 3, “Features” for details of each command.

3. Features

Command Format

  • Words in UPPER_CASE are the parameters to be supplied by the user e.g. in query p/POSTAL_CODE, POSTAL_CODE is a parameter which can be used as query p/760397.

  • Items in square brackets are optional e.g filter [f/] [n/] [ct/CARPARK_TYPE] can be used as filter f/ n/ ct/SURFACE ` or as `filter f/.

  • Parameters can be in any order e.g. if the command specifies filter [f/] [n/] [ct/CARPARK_TYPE], filter [n/] [f/] [ct/CARPARK_TYPE] is also acceptable.

3.1. Viewing help : help

Shows the list of available commands and how to use them Format: help

3.2. Query : query

Query for available car parks
Format: query p/POSTAL_CODE d/TRAVELING_DISTANCE

alias - q

Examples: * query p/612345 d/0.5 * q p/612345 d/0.5

3.3. Select car park : select

Select the car park you wish to go to.
Format: select CARPARK_NO

alias - s

Examples:

  • select HG83

  • s HG83

3.4. Filter car parks : filter

Filters the list of car parks by free parking and night parking, must query beforehand.
Format: filter [f/] [n/] [ct/CARPARK_TYPE]

Flags:

  • f - FREE_PARKING

  • n - NIGHT_PARKING

  • ct - CARPARK_TYPE

    • SURFACE

    • MUITISTORY

    • BASEMENT

Example:

  • filter f/ n/ ct/SURFACE

  • filter ct/BASEMENT

3.5. Sort car parks : sort

Sort the list of car parks by distance or name, must query beforehand.
Format: sort FILTER_TYPE

Flags:

  • DISTANCE

  • NAME

  • FP (Free Parking)

  • NP (Night Parking)

Example:

  • sort DISTANCE

3.6. Add favourite car parks or location: add

Add your favorite car park into your favorites list
Format: add c/CARPARK_NO

alias - a

Examples:

  • a c/SE12

  • add c/SE12

Add your favorite location into your favorites list
Format: add p/POSTAL_CODE

alias - a

Examples:

  • add p/612345

  • a p/612345

3.7. Show favorites list : list

Shows your favorites list, with all the car park details.
Format: list

3.8. Remove from favorites list : remove

Removes the specified carpark from your favorites list
Format: remove c/CARPARK_NO

alias - r

Examples:

  • remove c/SE12

  • r c/SE12

3.9. Data update : notify

Sets the interval in seconds to query from data.gov.sg
Format: notify TIME_SECONDS

Examples:

  • notify 60

4. FAQ

Q: How do I transfer my data to another computer?
A: Install the app in the other computer and overwrite the empty data file it creates with the file that contains the data of your previous car park book folder.

Q: If I do not know the postal code of the location, am I still able to find car parks through other means?
A: Yes, by typing keywords like street names.
